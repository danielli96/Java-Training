# Java Problem Training

A collection of Java solutions for LeetCode Easy and Medium problems, organized by topic and difficulty.

## Overview

This repository contains my solutions to some easy to medium problems in Java, focusing on writing clean, efficient, and well-documented code. Each solution includes time and space complexity analysis, along with multiple approaches where applicable.

## Setup

### Prerequisites
- Java 11 or higher
- Maven 3.6.X or higher

### Clone and Run

```bash
git clone https://github.com/danielli96/Java-Training.git
cd Java-Training

# Compile a solution
mvn clean compile

# Run tests (if using Maven)
mvn test

# Run specific test
mvn test -Dtest=TwoSumTest 
```
